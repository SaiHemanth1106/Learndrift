GETTING STARTED:
================

Option 1: Using Docker (Recommended)
------------------------------------
1. Start all services:
   docker-compose up

2. Access services:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:8000
   - API Docs: http://localhost:8000/docs

3. (Optional) Generate sample data:
   docker-compose exec backend python run_setup.py

Option 2: Local Setup (Without Docker)
--------------------------------------

Backend:
1. cd backend
2. python -m venv venv
3. source venv/bin/activate
4. pip install -r requirements.txt
5. cp .env.example .env
6. python run_setup.py
7. uvicorn app.main:app --reload

Frontend:
1. cd frontend
2. npm install
3. npm start

Database:
- Install PostgreSQL 15+
- Create database: createdb learndrift
- Load schema: psql learndrift < database/init.sql

API EXAMPLES:
=============

Create Student:
curl -X POST http://localhost:8000/api/students \
  -H "Content-Type: application/json" \
  -d '{"name": "John", "email": "john@uni.edu", "course_id": 1}'

Record Interaction:
curl -X POST http://localhost:8000/api/interactions \
  -H "Content-Type: application/json" \
  -d '{"student_id": 1, "question_id": 1, "user_answer": "A", 
       "is_correct": true, "time_taken_seconds": 45.5, 
       "retry_count": 0, "confidence": 0.9}'

Trigger Analysis:
curl -X POST http://localhost:8000/api/analysis/analyze/1

Get Alerts:
curl http://localhost:8000/api/analysis/alerts
